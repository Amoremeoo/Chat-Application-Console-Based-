# 💬 CHATO - Console Based Chat Application
CHATO is a console-based chat application built in C++ using Object-Oriented Programming and File Handling. It features user authentication, messaging, contact management, profile settings, conversation export, and an interactive colored console UI.

CHATO is a **Console-Based Chat Application** developed in **C++** using **Object-Oriented Programming (OOP)** concepts and **File Handling**.
The project provides a simple yet interactive messaging system where users can register, login, send messages, manage contacts, and customize their profiles — all within a beautifully designed console interface.

---

## 🚀 Features

* 🔐 User Registration & Login System
* 💬 Send & Receive Messages
* 📥 Inbox & Sent Messages
* 👥 Contact Management
* 🔍 Search Users by Name or ID
* 🛠 Profile & Settings Management
* 📂 File Handling for Data Storage
* 📜 Export Conversations to TXT File
* 🎨 Interactive Console UI with Colors
* ⚡ Fast Keyboard Navigation
* 🧠 Exception Handling & Secure Input

The application stores data using CSV/TXT files:

* `Users.csv` → User Accounts
* `Messages.csv` → Chat Messages
* `Chat_*.txt` → Exported Conversations

1. Open project in **VS Code** or any C++ IDE
2. Compile the program:

```bash

```

3. Run the executable:

```bash
./chato
```
